# 软件简介
EDEM-Fluent耦合接口编译及调试软件由“仿真小伙”开发，主要功能如下
* 搭建Fluent UDF编译及调试环境，**【不限制Fluent版本】**
* 搭建欧拉框架EDEM-Fluent耦合接口编译及调试环境，**【不限制EDEM版本和Fluent版本】**
* 搭建DPM框架EDEM-Fluent耦合接口编译及调试环境，**【不限制EDEM版本和Fluent版本】**
